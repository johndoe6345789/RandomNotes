# **Half-Life: Alyx — Quick Start Guide**

## **Requirements**

- Steam installed
    
- Meta Quest 2 or Quest 3 (via Meta Link or Air Link)
    
- Meta Link PC software installed: [https://www.meta.com/quest/setup/](https://www.meta.com/quest/setup/)
    
- USB‑C cable (for wired Link) or strong Wi‑Fi 6/6E (for Air Link)
    
- Room-scale or standing play space
    
- Updated GPU drivers (Vulkan recommended for AMD)
    

---

## **Launch Options (Optional but Recommended)**

### **Recommended**

```
-vulkan -console -vconsole
```

### **Useful Variants**

```
-vconsole          # External console window
-dx11             # Use DirectX 11 instead of Vulkan
```

### **Flat Mode (Not for first playthrough)**

```
-novr
```

---

## **Start a New Game**

1. Launch the game in VR
    
2. Point at **New Game**
    
3. Select difficulty
    
4. Choose your play style:
    
    - **Blink** (teleport)
        
    - **Shift** (quick short move)
        
    - **Continuous** (smooth locomotion)
        

---

## **Basic Controls**

### **Movement**

- **Left Stick** – Move
    
- **Right Stick** – Turn or smooth rotation
    
- **Grip Button** – Grab objects
    
- **Trigger** – Shoot / interact
    

### **Inventory**

- **Right Hand → Over-the-shoulder** – Ammo
    
- **Left Hand → Over-the-shoulder** – Health syringes
    

### **Gravity Gloves**

- Aim at object
    
- Flick wrist towards you
    
- Catch with grip button

## **Console Commands (Optional Cheats)**

Enable cheats first:

```
sv_cheats 1
```

### **Common Cheats**

```
god                 # Invincible
buddha              # Can’t die, stays at 1 HP
sv_infinite_ammo 1  # Unlimited ammo + mags
sv_infinite_clips 1 # Infinite magazines
noclip              # Fly through walls
impulse 101         # All weapons + full ammo
```

---

## **Useful Map Commands**

Skip menu and jump directly into a chapter:

```
map a1_intro_world
map a2_quarantine_entrance
map a3_hotel_lobby_basement
map a4_c17_zoo
map a5_vault
```

---

## **Tips for Smooth VR**

- Keep playspace clear
    
- Enable **vignettes** if motion-sensitive
    
- Use **continuous** movement once comfortable
    
- Adjust height in **Accessibility** menu
    
- If performance drops, disable MSAA or lower shadows
